# Output Artifacts

This folder contains supporting proof artifacts from the published article work.

## Files

- `article-desktop.png`
  Desktop screenshot of the published article.
- `article-mobile.png`
  Mobile screenshot of the published article.
- `article-validation.json`
  Validation notes from the article proof pass.

## Why these are here

These files are not the core commercial deliverables. They are included as supporting evidence that the article version of the recommendation was published and checked.
