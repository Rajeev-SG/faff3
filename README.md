# Faff3 AI Services Pack

This repository is a handoff pack for the faff3 team.

It brings together the market research, commercial positioning, partner-facing service packaging, and supporting proof created to help faff3 shape an AI implementation and consultancy offer.

## Start here

If you only read three things first, read them in this order:

1. `deliverables/partner-service-catalogue.md`
2. `deliverables/mckinsey-style-memo.md`
3. `research/market-findings.md`

## What is in this repo

### `deliverables/`

The main business outputs intended for discussion, refinement, and sharing.

- `partner-service-catalogue.md`
  A plain-English, partner-facing service catalogue with package names, positioning, and pricing bands.
- `mckinsey-style-memo.md`
  A strategy memo covering positioning, ICP, offer ladder, and a 90-day go-to-market plan.

### `research/`

The supporting research and interpretation behind the recommendations.

- `market-findings.md`
  Summary of the external market evidence and what it implies for faff3's commercial proposition.

### `output/`

Proof artifacts from the published article and validation pass.

- `article-desktop.png`
- `article-mobile.png`
- `article-validation.json`

### `meta/`

Repo navigation notes and publication context.

- `article-publication.md`
  Notes on the live article, where it was published, and where the source file lives outside this repo.

## Core recommendation in one sentence

Faff3 should position itself as the partner that helps businesses turn AI pilots into one clear, working result in a real workflow, then make that result repeatable.
